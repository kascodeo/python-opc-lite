Python-opc-lite
===============

|pol| is a light weight python library for Open Package Convention 
used in Open Office XML packages by |kas|


Overview
========

This library is used to deal with open office xml packages that follow 
Open Package Convention. 

It is used not only to read and write the packages but also to navigate through
the parts of the package by relationships
